⚠ Why Precision Is Low (and what to do about it)

This baseline uses synthetic data and a simple Gaussian model. In production, you'd improve it by:

1. Using real LOB data where spoofing patterns have real market impact signatures

2. Upgrading to a Mixture Density Network (MDN) that captures multi-modal price distributions

3. Adding time-to-cancel as a feature (spoofers cancel within 50-500ms)

4. Combining the profit test with the heuristic signals (size anomaly + cancel pattern)

5. Calibrating the fill probability model using actual historical fill rates per price level
